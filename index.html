My Portfolio

Online Course Unit

This is the first week of a massive open online course (MOOC) created by lecturers at ATU. My instructional design aim was to create a journey, to bring the learner from mentoring newbie to the point where they would be able to set up a mentorship for themselves. I added as much interactivity as possible, to give the student a sense of ownership on this course. Tools used included Premiere Pro, Veed, Adobe Illustrator, H5P, Articulate Storyline and Rise, and Microsoft Word. I also used Moodle to set up the MOOC on freecourses.atu.ie .

User guide

I developed this user guide while completing my Postgraduate Certificate in Information Design and Development at MTU. It’s aimed at an audience with minimal technical knowledge, to help writers create a Blogger-hosted static website. I delivered the content in multiple formats for different types of users: as video screencasts and as text content. This Blogger site (Websites for Writers) hosts the guide. The screencasts were created using Apple QuickTime and iMovie.

Storyline interaction

I created this in Articulate Storyline 360 as an interactive component of a marketing bridging course. There was a lot of information and I wanted to keep it in a single resource that the student could navigate logically. I decided to let them climb that mountain!
